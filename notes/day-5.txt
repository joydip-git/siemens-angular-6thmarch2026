routing:
---------------------
all routing related assets are in this package: @angular/router

simple types:
-------------------
individual route: Route type
	properties: path, component, redirectTo, pathMatch etc.

collection of routes: Routes (type Routes = Route[])
Route and Routes are simple types

directives:
--------------------
routerLink -> assign either a string path or an array of string path fragments
ex: routerLink="/products"
ex: [routerLink]="['/products','view',product.productId]"

services:
---------------
Router (engine) -> navigate, navigateUrl,.....
ex: this.router.navigate(['/products', 'edit', 2])

ActivatedRoute: 
Provides access to information about a route associated with a component that is loaded in an outlet. Use to traverse the RouterState tree and extract information from nodes.

built-in service class. instance of it stores current route (that you are navigating to) information. it is created by router and kept in router state (RouterState). a single instance of ActivatedRoute object is created for the very first time you visit a particular route and that same instance is reused over and again for further visits to the same route.

params property: An observable of the matrix parameters scoped to this route.
queryParams property: An observable of the query parameters shared by all the routes.


ActivatedRouteSnapshot: ITs NOT A SERVICE TYPE but a regular type
Contains the information about a route associated with a component loaded in an outlet "at a particular moment in time". ActivatedRouteSnapshot can also be used to traverse the router state tree.

ways to navigate:
	programmtical: use Router class's methods (navigate, navigateByUrl)
	declarative: use routerLink directive
